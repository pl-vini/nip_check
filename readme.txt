NIP Check

Aplikacj apozwala sprawdzić czy firmy są aktywnymi płatnikami VAT.
Dla Polski jest to API http://www.mf.gov.pl/uslugiBiznesowe/uslugiDomenowe/AP/WeryfikacjaVAT
Dla krejów UE to API http://ec.europa.eu/taxation_customs/vies/services/checkVatService

**demo.txt** przykładowy plik podlegający analizie

1. Instalacja node https://nodejs.org/en/
2. Komenda w katalogu z programem: npm install
3. Uruchomienie: node app nazwa_pliku_txt_nr_nip.txt
